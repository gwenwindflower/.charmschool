## Quick Install

```bash
# Back up existing config (if any)
mv ~/.config/zellij ~/.config/zellij.bak 2>/dev/null

# Create config directory
mkdir -p ~/.config/zellij/layouts

# Copy the config files
cp config.kdl ~/.config/zellij/
cp layouts/default.kdl ~/.config/zellij/layouts/

# (Optional) Download zjstatus locally for faster startup
mkdir -p ~/.config/zellij/plugins
curl -L https://github.com/dj95/zjstatus/releases/latest/download/zjstatus.wasm \
     -o ~/.config/zellij/plugins/zjstatus.wasm
```

If you download zjstatus locally, update the layout to use the local path:

```kdl
plugin location="file:~/.config/zellij/plugins/zjstatus.wasm" {
```

## First Launch

When you first start Zellij with zjstatus, it will ask for permissions. Press `y` to grant them:

- **ReadApplicationState** - needed for mode/tab info
- **ChangeApplicationState** - needed for frame toggling

## Customization

### Change timezone

In `layouts/default.kdl`, find:

```kdl
datetime_timezone "America/Los_Angeles"
```

Change to your timezone (e.g., `"America/New_York"`, `"Europe/London"`).

### Change date format

```kdl
datetime_format   "%H:%M"  // Just time
datetime_format   "%a %H:%M"  // Day + time
datetime_format   "%b %d %H:%M"  // Month day + time
```

### Want borders always visible?

In `config.kdl`:

```kdl
pane_frames true
```

And in the layout, set:

```kdl
hide_frame_for_single_pane "false"
```

## Troubleshooting

**Weird characters showing up?**
Set `simplified_ui true` in config.kdl

**Status bar not showing?**
Make sure you granted permissions when prompted, or restart Zellij

**Want to reset permissions?**
Clear the cache:

```bash
# Linux
rm -rf ~/.cache/zellij/

# macOS
rm -rf ~/Library/Caches/org.Zellij-Contributors.Zellij/
```
